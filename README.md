# Iris Flower Classification 🌸

## Project Description
This project uses Machine Learning to classify iris flowers into three species based on their measurements.

The model predicts the species of the iris flower using features like:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Dataset
The dataset used is the famous Iris dataset which contains 150 samples of iris flowers from three species:
- Setosa
- Versicolor
- Virginica

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

## Steps in the Project
1. Load the dataset
2. Data preprocessing
3. Train machine learning model
4. Test the model
5. Predict the flower species

## Output
The model predicts the species of the iris flower based on input features.
